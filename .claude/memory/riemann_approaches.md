## РАЗДЕЛ 2: ДЕТАЛИ КАЖДОГО ПОДХОДА

### A1 · Нулевые регионы (Zero-Free Regions)
**Идея:** Доказать, что ζ(s) ≠ 0 в полосе {Re(s) > ½ + δ} для некоторого δ > 0.
**Достигнуто [VERIFIED]:**
- Адамар + де ла Валле-Пуссен (1896): ζ(s) ≠ 0 при Re(s) > 1 - c/ln|t|
- Последующие улучшения: ln|t| → (ln|t|)^(2/3) · (ln ln|t|)^(1/3) (Виноградов–Коробов, 1958)
**Где застряло:** Убывание "1/ln|t|" — слишком медленное. Чтобы получить Re>½, нужен принципиально другой аргумент.
**Почему не сработало:** Метод не масштабируется — все улучшения дают лишь sub-exponential savings, не полосу.

---

### A2 · Теоремы о плотности (Density Theorems)
**Идея:** Оценить N(σ,T) — количество нулей с Re(s) > σ, Im(s) ≤ T.
**Достигнуто [VERIFIED]:**
- N(σ,T) = O(T^{A(1-σ)} log^B T) для разных A,B
- Лучший результат: A = 12/5 (при σ близко к 1)
**Где застряло:** Чтобы доказать N(½+ε, T) = 0, нужен A → 0, а это и есть ГР.
**Связь:** Density theorems дают условные результаты ("если ГР, то..."), а не само доказательство.

---

### A3 · Метод Харди (функция Z(t))
**Идея:** Функция Z(t) = ζ(½+it)·e^{iθ(t)} вещественна на Re=½. Нули Z(t) = нули ζ на критической прямой.
**Достигнуто [VERIFIED]:**
- Харди (1914): Z(t) меняет знак бесконечно часто → ∞ много нулей на Re=½
- Харди-Литтлвуд (1921): N₀(T) ≫ T (нулей на прямой пропорционально T)
**Где застряло:** "Бесконечно много" ≠ "все". Gap между N₀(T) ~ T и N(T) ~ T·ln(T)/2π.

---

### A4 · Метод Левинсона / Конри (доля нулей)
**Идея:** Оценить долю нулей N₀(T)/N(T) > c > 0.
**Достигнуто [VERIFIED]:**
- Левинсон (1974): c > 1/3 ≈ 33%
- Конри (1989): c > 2/5 = 40%
- Буи, Конри, Янг (2011): c > 41.05%
- Прэт (2013): c > 41.28%
**Где застряло:** Метод основан на вспомогательных функциях (mollifiers). Технически: оптимальный mollifier даёт ceiling ~40-42%. Новый принципиальный подход нужен для преодоления этого барьера.
**Ключевой вопрос [UNKNOWN]:** Существует ли mollifier или другой инструмент, дающий c > 50%?

---

### B1 · Гипотеза Гильберта-Пойа [OPEN, ПЕРСПЕКТИВНЫЙ]
**Идея:** Существует самосопряжённый (эрмитов) оператор H в гильбертовом пространстве, чьи собственные значения — мнимые части нулей ζ: λ_n = Im(ρ_n).
**Почему доказывало бы ГР:** Собственные значения эрмитова оператора — вещественны. Если λ_n = Im(ρ_n) вещественны и Re(ρ_n) = ½ из симметрии — ГР следует.
**Статус [INFERRED]:** Кандидаты на H предложены, но ни один строго не доказан.
**Прогресс:**
- Монтгомери (1973): пары нулей ζ ведут себя как GUE eigenvalue pairs — косвенное подтверждение
- Конри-Кац: операторы на функциональных полях (Вейль) — аналог работает
- Берри-Китинг (1999): H = xp квантового хаоса — интуиция, не доказательство
**Почему сложно:** Нет конструктивного способа найти H. Мы знаем, что если он есть — ГР верна. Но как его построить?

---

### B2 · Random Matrix Theory (RMT)
**Идея:** Статистика нулей ζ совпадает со статистикой собственных значений случайных матриц из GUE (Gaussian Unitary Ensemble).
**Достигнуто [VERIFIED]:**
- Монтгомери (1973): формула корреляции пар нулей = r₂ GUE
- Одлыко (1987): вычислительное подтверждение на 10⁶ нулях
- Кац-Зарнак (1999): GUE = "правильная" группа симметрии для ζ
**Где застряло:** Статистическое сходство наблюдается, но не доказано аналитически. RMT даёт предсказания (подтверждённые), но не механизм доказательства.
**Связь с B1:** RMT — это эмпирическое свидетельство существования оператора Гильберта-Пойа.

---

### B3 · Квантовый хаос
**Идея:** Нули ζ ~ квазиуровни энергии хаотичной квантовой системы (например, бильярд).
**Представители:** Берри, Китинг, Боас
**Статус [WEAK]:** Много красивых аналогий, формального доказательства нет.
**Полезность:** Физическая интуиция помогает строить конъектуры, но не заменяет доказательство.

---

### C1 · Функциональные поля (Вейль, 1948) [VERIFIED ✓ — НО АНАЛОГ]
**Идея:** Доказать аналог ГР для дзета-функций алгебраических кривых над конечными полями.
**Достигнуто [VERIFIED]:** Вейль (1948) доказал аналог ГР для кривых: все нули |в полосе| Re(s) = ½.
**Важно:** Это НЕ сама ГР, это аналог в другом (более простом) мире. Алгебраические методы там работают, потому что конечное поле даёт алгебраическую структуру.
**Почему не переносится:** ℤ — не конечное поле. Переход от функциональных полей к числам — ключевая open problem (программа Ленглэндса).
**Что взять:** Структура доказательства Вейля — потенциальный blueprint.

---

### C2 · Теория мотивов / Программа Ленглэндса [OPEN, ДОЛГОСРОЧНЫЙ]
**Идея:** ζ(s) — часть L-функции мотива над ℚ. Доказать ГР через общую теорию L-функций.
**Статус [INFERRED]:** Программа Ленглэндса включала бы ГР как частный случай, но сама программа не завершена.
**Горизонт:** Математики оценивают это как проект на 50-100 лет.

---

### D1 · Критерий Робена (1984) [VERIFIED — эквивалентность]
**Утверждение:** ГР ⟺ σ(n) < e^γ · n · ln(ln(n)) для всех n ≥ 5041
**Прогресс:** Проверено вычислительно до n ~ 10^13.
**Почему интересно:** Арифметическое утверждение о делителях — другой язык для ГР.
**Где застряло:** Доказать неравенство для всех n — столь же сложно, как ГР.

---

### D2 · Критерий Ли (1997) [VERIFIED — эквивалентность]
**Утверждение:** ГР ⟺ все коэффициенты Ли λ_n ≥ 0 для n = 1, 2, 3, ...
**Где λ_n:** производные логарифма ξ-функции в определённой точке
**Прогресс:** λ_1 > 0 доказано, λ_n > 0 для n≤10^5 — вычислительно.
**Почему интересно:** Позитивность коэффициентов — может быть доказана через другие техники?

---

### D4 · Теория следов Зельберга (1956) [VERIFIED для компактных поверхностей]
**Идея:** Для компактных гиперболических поверхностей: формула следов связывает собственные значения лапласиана с длинами замкнутых геодезик — аналог явной формулы Римана.
**Статус:** Работает для компактного случая. Для ζ(s) — некомпактная ситуация, технически сложнее.
**Связь:** Это аналитический аналог Гильберта-Пойа в компактном случае.

## РАЗДЕЛ 3: ОШИБОЧНЫЕ "ДОКАЗАТЕЛЬСТВА" — ПАТТЕРНЫ

### Типичные ошибки в ложных доказательствах

**Паттерн 1: Замена "почти все" на "все"**
- Доказывается N₀(T)/N(T) → 1 при T → ∞ (ещё не доказано!)
- Ошибочно заключается: "значит все нули на прямой"
- Проблема: предел 1 не достигается за конечное время

**Паттерн 2: Цикличное использование ГР**
- Где-то в доказательстве неявно предполагается ГР
- Например, использование оценок, верных "при условии ГР"

**Паттерн 3: Аналогия вместо доказательства**
- "ГР верна для функциональных полей (Вейль) → верна для ζ(s)"
- Разница между двумя ситуациями не устранена

**Паттерн 4: Компьютерные "доказательства"**
- "Проверено 10^13 нулей → доказано для всех"
- Аналог: "все лебеди которых я видел — белые → чёрных лебедей нет"

**Паттерн 5: Подмена задачи**
- Доказывается эквивалентное утверждение, НО само утверждение остаётся открытым
- "Robin's inequality верна для n ≤ 10^13 → ГР верна" (НЕТ)

### Что считается наиболее перспективным [WEAK — based on community signals]

- Попытки пробить ceiling в 41-42%
- Новые mollifiers, вспомогательные функции
- [INFERRED] Пракаш Беллапосе и группы в MIT/Cambridge активны здесь

**Направление 2: Оператор Гильберта-Пойа через квантовую механику**
- Конри, Берри, Китинг (Bristol) — активная группа
- Ищут конкретный самосопряжённый оператор
- Связь с моделями квантового хаоса

**Направление 3: Geometric Langlands completion**
- После доказательства геометрической программы Ленглэндса (Фарго-Разумнов и др., 2024)
- Возможное приближение к числовому случаю
- [WEAK] Теренс Тао: "Это многообещающее направление, но горизонт далёк"

**Направление 4: Новые аналитические техники**
- Разработки в теории экспоненциальных сумм (Bourgain, Watt)
- Улучшения оценок нулевых регионов

### Что НЕ работает по мнению сообщества [INFERRED]
- Прямые вычислительные попытки (нет конечного числа нулей)
- "Элементарные" доказательства (ГР — не элементарная задача)
- Аналогии без формального перехода от функциональных полей

## ИТОГ: ДИАГНОЗ

**Почему ГР не решена за 165 лет:**
1. Задача стоит на пересечении 4 областей (анализ / алгебра / геометрия / физика) — нет единого инструмента
2. Вычислительных свидетельств много, структурного понимания — мало
3. Каждый подход заходит в тупик у одного и того же барьера: от "почти все" к "все"
4. Аналог (Вейль, 1948) доказан, но перенос требует математики, которой ещё нет

**Самые жизнеспособные атаки на 2026:**
- A-07 (Левинсон) + B1 (Гильберта-Пойа) + B2 (RMT) — три взаимосвязанных направления
