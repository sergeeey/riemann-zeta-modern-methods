## Детали атомов

### A-01 · Функциональное уравнение [VERIFIED ✓]
**Утверждение:** ζ(s) = 2^s · π^{s-1} · sin(πs/2) · Γ(1-s) · ζ(1-s)
**Доказал:** Риман, 1859
**Почему важно:** создаёт симметрию относительно Re(s)=½; если ρ — ноль, то и 1-ρ — ноль
**Для нас:** фундамент, объясняет ПОЧЕМУ ½ — "правильное" место
**Эксперимент:** не нужен, доказано

---

### A-02 · Тривиальные нули [VERIFIED ✓]
**Утверждение:** ζ(-2n) = 0 для n = 1, 2, 3, ...
**Механизм:** sin(πs/2) = 0 в точках s = -2, -4, -6, ...
**Для нас:** закрытый вопрос, используем как базовый пример верификации
**Эксперимент:** Python проверка (быстрый контроль)

---

### A-03 · Критическая полоса [VERIFIED ✓]
**Утверждение:** Все нетривиальные нули: 0 < Re(s) < 1
**Доказали:** Адамар + де ла Валле-Пуссен, 1896 (независимо)
**Следствие:** Теорема о простых числах π(x) ~ x/ln(x)
**Для нас:** граница сужена, но Re(s)=½ внутри этой полосы ещё не доказано
**Эксперимент:** не нужен, доказано

---

### A-04 · Бесконечно много нулей на Re=½ [VERIFIED ✓]
**Утверждение:** |{ρ : ζ(ρ)=0, Re(ρ)=½, |Im(ρ)|≤T}| → ∞ при T→∞
**Доказал:** Харди, 1914
**Метод:** через функцию Харди Z(t), которая вещественна на критической прямой
**Для нас:** "некоторые лежат" ≠ "все лежат" — gap здесь
**Эксперимент:** A-04-hardy-Z-function (реализация Z(t), визуализация)

---

### A-05 · Связь с простыми числами [VERIFIED ✓]
**Явная формула:** ψ(x) = x - Σ_ρ x^ρ/ρ - ln(2π) - ½ ln(1-x^{-2})
**Теорема Фон Коха (1901):** ГР ⟺ |π(x) - Li(x)| < (1/8π) √x ln(x)
**Для нас:** ключевое — ГР = точная оценка ошибки в π(x)
**Эксперимент:** A-05-prime-error (вычислить реальную ошибку π(x)-Li(x), сравнить с O(√x))

---

### A-06 · Эквивалентные формулировки [VERIFIED ✓]
**Robin (1984):** σ(n) < e^γ · n · ln(ln(n)) для всех n ≥ 5041 ⟺ ГР
**Li (1997):** все коэффициенты Ли λ_n > 0 ⟺ ГР
**Для нас:** альтернативные пути — может, одну из них легче доказать?
**Эксперимент:** A-06-robin (проверить Robin inequality вычислительно до N=10^7)

---

### A-07 · Доля нулей на Re=½ [PARTIAL ~41%]
**Текущий рекорд:** >41.28% (Прэт, 2013)
**Подход:** метод Левинсона (1974) + улучшения
**Gap до 100%:** огромный — от 41% до "все" нет мостика
**Для нас:** АКТИВНАЯ ОБЛАСТЬ. Каждый процент = прорыв
**Эксперимент:** A-07-density (изучить метод Левинсона, понять где ceiling)

---

### A-08 · Полоса без нулей вблизи Re=1 [PARTIAL]
**Доказано:** ζ(s) ≠ 0 при Re(s) > 1 - c/ln(|t|+2)
**Надежда:** расширить эту полосу до Re(s) > ½ + ε — это докажет ГР
**Для нас:** возможная атака: widening zero-free region
**Эксперимент:** A-08-zero-free (изучить методы расширения полосы)

---

### A-09 · Random Matrix Theory и статистика нулей [PARTIAL]
**Открытие:** статистика нулей ζ(s) совпадает со статистикой GUE (Монтгомери-Одлыко)
**Статус:** совпадение наблюдается, но НЕ доказано строго
**Для нас:** физический/вероятностный подход — нули "ведут себя" как собственные значения
**Связь:** это база для подхода Гильберта-Пойа
**Эксперимент:** A-09-rmt (вычислить статистику пар нулей, сравнить с GUE)

---

### A-10 · Вычислительная верификация [VERIFIED-REAL ✓]
**Факт:** Первые 2×10¹³ нулей — все на Re=½
**Инструменты:** mpmath, LMFDB, Odlyzko datasets
**Для нас:** база для тестов, но НЕ доказательство
**Эксперимент:** A-10-verify (загрузить данные Одлыко, визуализировать, подтвердить)

---

### A-11 · САМА ГИПОТЕЗА [OPEN ❓]
**Утверждение:** ∀ρ: ζ(ρ)=0 и ρ нетривиален → Re(ρ) = ½
**Статус:** открыта с 1859 года
**Для нас:** финальная цель, прямая атака только после изучения A-07, A-08, A-12
**Эксперимент:** synthesis (объединение результатов других атомов)

---

### A-12 · Подход Гильберта-Пойа [OPEN ❓]
**Идея:** Существует самосопряжённый оператор H, чьи собственные значения = мнимые части нулей ζ
**Статус:** гипотеза о гипотезе, не доказана, но мощная
**Связь:** собственные значения эрмитового оператора — вещественны → Re(s) = ½
**Современный прогресс:** связь с Random Matrix Theory, операторы из физики
**Для нас:** возможный путь к доказательству через спектральную теорию
**Эксперимент:** A-12-hilbert-polya (изучить существующие кандидаты на оператор H)

---

### A-13 · Обобщённая ГР (GRH) [OPEN ❓]
**Утверждение:** все нули L-функций Дирихле лежат на Re(s) = ½
**Статус:** ещё сложнее, чем ГР
**Для нас:** не фокус на этом этапе

---

### A-14 · Геометрический/Физический подход [OPEN ❓]
**Включает:**
- Подход Гранта (Eisenstein lattice, iHarmonic) — [HYPOTHESIS, требует аудита]
- Квантовый хаос и билиарды
- Теория следов Зельберга как аналог
**Для нас:** Phase 4 (после аудита A-01..A-13)
