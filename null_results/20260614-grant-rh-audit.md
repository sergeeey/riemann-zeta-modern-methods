## Фатальные узлы (почему REJECT)

### Doc A — §9 Step 4: КАТЕГОРИАЛЬНАЯ ОШИБКА (🔴 CRITICAL)
Подменяются два разных «O(√x)»:
- (a) ошибка подсчёта точек решётки Эйзенштейна (задача Гаусса о круге, Thm 6.3) — реально O(√x), элементарно, **верно**;
- (b) осцилляция явной формулы E(x)=Σ_ρ x^ρ/ρ — её оценка O(√x) **ЕСТЬ САМА RH** (von Koch 1901).

Слово «boundary» обслуживает оба смысла (граница решётки ↔ граничное scaling) → equivocation. Шаг «E(x) ограничена ёмкостью границы решётки» = **assuming the conclusion**. Моста (a)→(b) не существует — это разные функции.
Дополнительно: «½ из dim(boundary)/dim(bulk)» (§6.2) — exponent периметра √x (общий для ЛЮБОЙ 2D-решётки) отождествлён с Re(s)=½ только через общую цифру ½; вывода нет (симметрию ½ и так даёт функц. уравнение).

### Doc B — Conj 17.12 Spectral Isomorphism (🔴 CRITICAL)
Это незакрытый разрыв Гильберта–Пойи: оператор 𝓡 даёт КОНЕЧНОЕ число mode-eigenvalues (Thm 17.7: Mode 3 → ровно 3 собств. знач.), нулей ζ — БЕСКОНЕЧНОСТЬ. Биекция Spec(𝓡)↔{ρ} — **Conjecture**, Thm 17.13 = «If isomorphism holds, then RH».
**Автор сам признаёт неполноту** (стр. 87): *«The gap between O(√n) (proved) and O(1) (empirical) is the remaining frontier.»* Bounded Deviation (Conj 17.24, |Δₙ|≤0.6075) проверена лишь на 27 станциях.
Циркулярность t(n): R(n) считают Ньютоном из *известных* π(10ⁿ) (стр. 77) — измерение остатка после fit, не предсказание (ср. защитный заголовок §13 «Python Implementation (Non-Circular)»).
