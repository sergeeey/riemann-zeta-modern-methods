# sandbox INDEX — реестр сырых гипотез

Каждая гипотеза **ОБЯЗАНА** иметь вердикт. Гипотеза без вердикта = долг.

## Легенда вердиктов
- `raw` — только закинута, Zero-Signal Gate ещё не прогнан
- `promote` — прошла гейт → перенесена в `experiments/<id>/` (полный FL)
- `parked` — валидна, но отложена (условие возврата указано в `note.md`)
- `drop` — не прошла гейт / опровергнута (след остаётся, не удаляем)
