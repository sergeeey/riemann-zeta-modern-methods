# Estimand — A-07 · Levinson density + Speiser test
Date: 2026-06-14
EstimandOps L0 + L1

---

## L0: Question Type
**Descriptive.**
Вопрос: "Где расположены нули ζ'(s) относительно критической прямой (тест Шпайзера,
⟺ RH), и почему метод Левинсона доказывает лишь ~41% нулей на прямой, а не 100%?"
- НЕ Causal, НЕ доказательство RH.

---

## Контекст (lit-review 2026-06-14, см. review_levinson_speiser.md)
- Метод Левинсона: доля нулей ζ на Re=½. Цепочка рекордов:
  Levinson 1/3 (1974) → Conrey 2/5 (1989) → BCY 41.05% (2011) → Feng ~41.07% (2012, спорно)
  → **PRZZ 41.7% (>5/12, 2020) — текущий рекорд**.
- Ceiling ~41%: структурная потеря в полоске ½≤σ<½+a/lnT (нули ζ' у самой прямой,
  Goldston–Suriajaya 2025 "inherent loss"). θ=4/7 — граница сумм Клоостермана, не фундамент.
- **Speiser (1934): RH ⟺ ζ'(s) не имеет нулей в 0<Re(s)<½.**
- Levinson–Montgomery (1974): N₁⁻(T) = N⁻(T) + O(lnT) — основа метода.

## L1: Estimand Attributes

| Attribute | Specification |
|-----------|---------------|
| **Population** | Нули ζ'(s) в критической полосе, высоты Im ∈ (0, 50] |
| **Intervention** | (a) argument principle: счёт нулей ζ' в [δ, ½]×(0,T] (СЛЕВА); (b) findroot по сетке: локализация нулей ζ' справа |
| **Comparator** | Speiser: слева ожидается 0 (⟺ RH); первые 5 нулей ζ' справа — эталон из lit-review |
| **Endpoint** | число нулей ζ' слева от прямой; координаты (Re,Im) нулей справа |
| **Summary Measure** | count_left (должен=0); max совпадение первых 5 нулей с эталоном; min Re найденных |
| **MCID** | ЛЮБОЙ ноль ζ' с Re<½ = нарушение Speiser = опровержение RH (до этой высоты) |

### ICE
- **Контур ∮ для нулей СПРАВА нестабилен** (полюса ζ''/ζ' в нулях, lit-review дал 4.06 вместо 5)
  → strategy: для счёта справа использовать findroot по сетке, НЕ контур. Контур только слева (устойчив).
- **Тривиальные нули ζ'** (между тривиальными нулями ζ, Re<0) → strategy: исключить, брать δ>0, Im>0.

## Natural Language Statement (ДО результатов)
> Мы считаем число нулей ζ'(s) слева от критической прямой (0<Re<½, 0<Im≤50)
> через принцип аргумента и локализуем нули ζ' справа через findroot по сетке,
> проверяя согласие с теоремой Шпайзера (слева=0 ⟺ RH) и воспроизводя первые
> пять нулей ζ', обрабатывая нестабильность контура справа переходом на findroot.

---

## What This Result Does NOT Mean (≥3, ДО результатов)
1. **НЕ доказывает RH.** Speiser-проверка до Im=50 — эмпирика, как и счёт самих нулей.
2. **НЕ улучшает 41%.** Мы иллюстрируем механизм Левинсона, не двигаем границу.
3. **НЕ объясняет, КАК пробить 41%** (это открытая проблема — нужна новая идея, не mollifier).
4. Эмпирически доля нулей на прямой = 100%; Левинсон доказывает 41%. Разрыв = граница ДОКАЗУЕМОГО, не наблюдаемого.

Descriptive → argument principle (слева) + findroot grid (справа). Без модели.

## Sensitivity (≥2)
1. **Высота:** счёт нулей ζ' слева на (0,25] и (0,50] — оба должны быть 0 (стабильность Speiser).
2. **Полоса контроля:** счёт нулей ζ' справа [½,3]×(0,50] — должно быть >0 (нетривиальные нули существуют), сверить с findroot.
